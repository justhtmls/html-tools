JWT 解码器

解码和验证 JSON Web Token

关于这个工具

JWT (JSON Web Token) 是一种用于双方之间传递安全信息的开放标准 (RFC 7519)。这个工具可以帮你解码 JWT 并查看其内容，包括头部、负载和签名部分。

功能特点

  • 实时解码 JWT Token
  • 显示 Header、Payload、Signature 三部分
  • 检查 Token 过期时间
  • 格式化 JSON 输出
  • 一键复制各部分内容
  • 完全在浏览器本地运行，保护隐私

使用场景

  • 调试 API 认证问题
  • 查看 Token 中的用户信息
  • 学习 JWT 结构
  • 检查 Token 是否过期
  • 分析 JWT 内容

注意事项

此工具仅进行 Base64 解码，不会验证签名。请不要将敏感的 Token 粘贴到任何在线工具中。
